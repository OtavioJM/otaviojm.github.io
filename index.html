HTML Cheat Sheet

Essential HTML elements and their usage

1. <!DOCTYPE html>

Declares the document type and HTML version. Always the first line in an HTML document.

2. <html>

Root element that wraps all content on the page. Contains lang attribute for language.

3. <head>

Contains meta information about the document (title, links to CSS, charset, etc.)

4. <meta charset="UTF-8">

Specifies character encoding for the HTML document. UTF-8 supports all languages.

5. <title>

Sets the title shown in browser tab. Important for SEO and user experience.

6. <body>

Contains all visible content: text, images, links, tables, lists, etc.

7. <h1> to <h6>

Heading tags. h1 is largest/most important, h6 is smallest. Use hierarchically.

8. <p>

Paragraph tag. Used for blocks of text. Automatically adds space before and after.

9. <a href="">

Anchor tag for links. href attribute specifies destination URL. Can link to pages or sections.

10. <img src="" alt="">

Image tag. src is file path, alt is description for accessibility. Self-closing tag.

11. <div>

Division/container element. Block-level. Used to group elements for styling or layout.

12. <span>

Inline container for text. Used to style part of text without breaking flow.

13. <ul> and <li>

Unordered list with list items. Creates bullet points. Good for non-sequential items.

14. <ol> and <li>

Ordered list with list items. Creates numbered list. Use for sequential steps.

15. <table>, <tr>, <td>

Table elements: table container, table row, table data cell. Use for tabular data.

16. <form>

Container for input elements. action attribute sets where to send data, method sets how.

17. <input type="">

Input field. Type can be text, email, password, number, checkbox, radio, submit, etc.

18. <button>

Clickable button. Can be used in forms or with JavaScript for interactions.

CSS Cheat Sheet

Essential CSS properties and techniques

1. color

Sets text color. Values: color names, hex (#ff0000), rgb(255,0,0), or hsl(0,100%,50%).

2. background-color

Sets background color of element. Same value types as color property.

3. font-size

Sets text size. Units: px (pixels), em (relative to parent), rem (relative to root), % (percentage).

4. font-family

Sets font type. List multiple fonts as fallbacks. Use quotes for multi-word fonts.

5. margin

Space outside element border. Can set all sides or individually (margin-top, margin-right, etc.)

6. padding

Space inside element border. Creates breathing room around content. Same syntax as margin.

7. border

Creates border around element. Syntax: width style color (e.g., 2px solid black).

8. display

Controls element layout. Common values: block, inline, inline-block, flex, grid, none.

9. position

Sets positioning method. Values: static (default), relative, absolute, fixed, sticky.

10. flexbox (display: flex)

Modern layout system. Properties: justify-content, align-items, flex-direction, gap.

11. grid (display: grid)

2D layout system. Properties: grid-template-columns, grid-template-rows, gap.

12. width and height

Sets element dimensions. Can use px, %, vw/vh (viewport units), or auto.

13. transition

Animates property changes. Syntax: property duration timing-function delay.

14. :hover pseudo-class

Applies styles when mouse hovers over element. Great for interactive effects.

15. box-shadow

Adds shadow to element. Syntax: h-offset v-offset blur spread color.

16. text-align

Aligns text horizontally. Values: left, right, center, justify.

17. @media queries

Creates responsive designs. Example: @media (max-width: 768px) for mobile styles.

18. z-index

Controls stacking order of positioned elements. Higher values appear on top.

JavaScript Cheat Sheet

Essential JavaScript concepts and methods

1. Variables (let, const, var)

Store data. let and const are block-scoped. const can't be reassigned. Avoid var.

2. console.log()

Prints output to browser console. Essential for debugging and testing code.

3. Functions

Reusable code blocks. Syntax: function name(parameters) { code } or arrow: (params) => { code }

4. getElementById()

Selects HTML element by id attribute. Returns single element or null if not found.

5. querySelector()

Selects first element matching CSS selector. More flexible than getElementById.

6. addEventListener()

Attaches event handler to element. Syntax: element.addEventListener('event', function).

7. if/else statements

Conditional logic. Executes code based on whether condition is true or false.

8. for loops

Repeats code a specific number of times. Syntax: for(let i=0; i<10; i++) { code }

9. Arrays

Ordered list of values. Created with []. Access with index: array[0]. Has methods like push, pop.

10. Objects

Store key-value pairs. Created with {}. Access with dot notation: object.key or object['key'].

11. innerHTML

Gets or sets HTML content of element. Example: element.innerHTML = '<p>New content</p>'.

12. classList

Manipulates CSS classes. Methods: add(), remove(), toggle(), contains().

13. setTimeout()

Executes code after delay. Syntax: setTimeout(function, milliseconds).

14. setInterval()

Repeatedly executes code at intervals. Syntax: setInterval(function, milliseconds).

15. Math object

Mathematical operations. Methods: Math.random(), Math.floor(), Math.round(), Math.max().

16. Date object

Work with dates and times. Create: new Date(). Methods: getFullYear(), getMonth(), getDate().

17. JSON.parse() and JSON.stringify()

Convert between JSON and JavaScript objects. parse() converts string to object, stringify() does opposite.

18. try/catch

Error handling. try block runs code, catch block handles errors if they occur.
